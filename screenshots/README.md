# Screenshots

This directory contains screenshots of the SaaS landing page.

## How to Add Screenshots

1. Run the development server: `npm run dev`
2. Take screenshots of:
   - Light mode (full page)
   - Dark mode (full page)
   - Hero section
   - Features section
   - Pricing section
   - Testimonials section
   - Mobile view

3. Name them appropriately:
   - `light-mode.png`
   - `dark-mode.png`
   - `mobile-view.png`
   - etc.

4. Place them in this directory

## Recommended Tools

- **macOS:** Cmd + Shift + 4
- **Windows:** Snipping Tool or Win + Shift + S
- **Browser Extensions:** Full Page Screen Capture
- **Online Tools:** Screely.com for beautiful mockups
